RESERVED[3:0] CLK_SEL PLL_STATUS PLL_FUNCTION SI571_OE
  • SI571_OE [read/write]: Si 571 Output Enable
    Si571 output enable pin, check datasheet for proper voltage standard and signal polarity
  • PLL_FUNCTION [read/write]: AD9510 PLL function
    AD9510 function pin (behaviour depends on configuration of chip, output from FPGA)
  • PLL_STATUS [read-only]: AD9510 PLL Status
    AD9510 status pin - for monitoring PLL status and sync (behaviour depends on configuration of chip, input to FPGA)
  • CLK_SEL [read/write]: Reference Clock Selection
    TS3USB221 clock select (for FMC REFIN line)
    0 - clock from external source (MMCX J4)
    1 - clock from FPGA (FMC_CLK line)
  • RESERVED [read-only]: Reserved
    Ignore on read, write with 0's.
