7 6 5 4 3 2 1 0
- - - - RCV_SRC TRANSM_COUNT_RST RCV_COUNT_RST DIR
  • DIR [read/write]: Trigger Direction
    1 - Receiver Mode
    0 - Transmitter Mode
  • RCV_COUNT_RST [read/write]: Receiver Pulse Counter Reset
    1 - Reset
    0 - Normal
  • TRANSM_COUNT_RST [read/write]: Transmitter Pulse Counter Reset
    1 - Reset
    0 - Normal
  • RCV_SRC [read/write]: Receiver Source
    1 - Internal Signals
    0 - Triggers
  • RCV_IN_SEL [read/write]: Select Receiver Input
    Select input that will be used by the receiver
  • TRANSM_OUT_SEL [read/write]: Select Transmitter Output
    Select output that will be used by the transmitter
