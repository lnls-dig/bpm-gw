  • FSM_START_ACQ [write-only]: State machine acquisition_start command (ignore on read)
    1: ACQ_START (start acquisition, only when FSM is idle)
  • FSM_STOP_ACQ [write-only]: State machine stop command (ignore on read)
    1: ACQ_STOP (stop acquisition, anytime)
  • RESERVED1 [read/write]: Reserved1
    Ignore on read, write with 0's
  • FSM_ACQ_NOW [read/write]: Acquire data immediately and don't wait for any trigger (ignore on read)
    1: Acquire immediately
    0: Wait for trigger
  • RESERVED2 [read/write]: Reserved2
    Ignore on read, write with 0's

    0: not full
