  • FMC_IDELAY_RST [read/write]: FMC_IDELAY_RST
    Reset IDELAY components in FPGA (for data transmission from ISLA ADC chips). Need to calibrate IDELAY lines. 0 - reset off, 1 - reset on
  • FMC_FIFO_RST [read/write]: FMC_FIFO_RST
    Reset FIFOs in FPGA (data aquisition from ISLA ADC chips). Useful for calibration. 0 - reset off, 1 - reset on
  • FMC_IDELAY0_RDY [read-only]: FMC_IDELAY0_RDY
    Calibration status of IDELAY lines for ADC0 chip
  • FMC_IDELAY1_RDY [read-only]: FMC_IDELAY1_RDY
    Calibration status of IDELAY lines for ADC1 chip
  • FMC_IDELAY2_RDY [read-only]: FMC_IDELAY2_RDY
    Calibration status of IDELAY lines for ADC2 chip
  • FMC_IDELAY3_RDY [read-only]: FMC_IDELAY3_RDY
    Calibration status of IDELAY lines for ADC3 chip
  • RESERVED1 [read-only]: Reserved
    Ignore on read, write with 0's.
  • TEMP_ALARM [read-only]: Temperature Alarm
    Temperature alarm from LM75 chips
  • RESERVED2 [read-only]: Reserved
    Ignore on read, write with 0's.

HW address: 0xa

RESERVED[2:0] ADC_RESET ADC_STATUS0 ADC_DONE ADC_PHASE ADC_EN
  • ADC_EN [read/write]: ADC_DCM
    Enable one phase shift for ADC DCM
  • ADC_PHASE [read/write]: ADC_PHASE_INC
    ADC phase shift: 1 - increment, 0 - decrement
  • ADC_DONE [read-only]: ADC_DCM_DONE
    Phase shifting done (ADC DCM): 1 - done, 0 - in progress
  • ADC_STATUS0 [read-only]: ADC_DCM_STATUS0
    Status of phase shifting (ADC DCM, status bit 0): 1 - error, outside the limits, 0 - ok
  • ADC_RESET [read/write]: ADC_RESET
    Reset ADC DCM (should be done on startup)
  • RESERVED [read-only]: Reserved
    Ignore on read, write with 0's.
